# Supplementary Appendix

This repository provides the supplementary appendix for the paper:

## Contents

- [Appendix](docs/appendix.md)

## Overview

The appendix provides additional implementation and evaluation details, including:

1. Direct VLM prompting templates.
2. JSON repair and parsing rules.
3. Coordinate normalization rules.
4. Duplicate removal and NMS rules.
5. Soft-NMS validation results.
6. Inria BoxRefine diagnostic discussion.
7. Locked inference hyperparameters.

## Notes

This repository only contains supplementary appendix materials. It does not include training data, private annotations, or the full paper manuscript.

If the paper is under double-blind review, please use an anonymized GitHub account or an anonymous repository service before sharing this link with reviewers.
